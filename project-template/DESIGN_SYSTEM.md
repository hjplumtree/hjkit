
# Design System

## Colors
Use semantic tokens:
background, surface, text, muted, border, primary, accent

## Typography
Title / Section / Body / Meta
Clear hierarchy

## Spacing
4 / 8 / 12 / 16 / 24 / 32

## Components
Buttons → clean, clear states
Cards → light, not crowded
Inputs → minimal, readable

## Style
Modern, structured, calm

## Avoid
Flat UI, generic SaaS, too many cards
